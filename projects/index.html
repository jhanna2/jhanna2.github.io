CTI Agent

A Go-based cyber threat intelligence pipeline that pulls external threat bulletins, enriches IOCs in parallel against free feeds, triages findings against a live cluster asset inventory in OpenSearch, and uses a locally-hosted LLM for context-aware analysis. Content Sanitizer & TOON Normalizer

A Python library for cleaning and compacting external content before it reaches an LLM. The sanitizer removes HTML attack surface, invisible text, Unicode tricks, and embedded prompt injection. The TOON normalizer compacts the result to semantic core — around 95% token reduction on typical web content while preserving structured data like IOCs. Built from real experimentation with HTML injection and image steganography.
