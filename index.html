Jeff Hanna

Senior Security Engineer at GoDaddy, where I've spent the last three years building the programs, pipelines, and automation that didn't exist before I got there.

The work ranges from LLM-orchestrated threat intelligence systems and credential remediation pipelines to DNS-scale packet analysis and SOC-style triage workflows — most of it built from scratch, in a domain where the tools rarely do exactly what you need. Before GoDaddy: intelligence program development at U.S. Cyber Command and the Washington Army National Guard, including USCYBERCOM's first OSINT program.

The through-line is the same everywhere: find the gap, build the solution, make it robust enough to outlast you.

The homelab projects below skew the same way: detection you can prove in-cluster, intel pipelines you can trace end-to-end, and LLM guardrails that default closed — not slide-deck security.

Content Sanitizer & TOON Normalizer

A Python library for cleaning and compacting external content before it reaches an LLM. The sanitizer removes HTML attack surface, invisible text, Unicode tricks, and embedded prompt injection. The TOON normalizer compacts the result to semantic core — around 95% token reduction on typical web content while preserving structured data like IOCs.
